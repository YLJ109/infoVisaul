# 计算机相关岗位数据可视化系统实训报告

## 1. 实训目的

本次实训旨在通过构建一个计算机相关岗位数据可视化系统，帮助学生掌握以下技能：

1. **数据处理能力**：学习如何清洗和处理大规模招聘数据，包括缺失值处理、数据格式标准化等。
2. **数据可视化技术**：掌握使用PyECharts进行多种图表类型的绘制，包括桑基图、地理分布图、饼图、折线图、柱状图和热力图等。
3. **GUI应用开发**：熟练运用PyQt6框架开发桌面应用程序，实现用户友好的图形界面。
4. **全栈开发思维**：从数据采集、处理到可视化展示的全流程实践，培养完整的项目开发能力。
5. **实际问题解决能力**：通过分析真实的计算机岗位数据，洞察就业市场趋势，为职业规划提供数据支撑。

## 2. 实训环境

### 2.1 硬件环境
- 处理器：Intel Core i5 或更高配置
- 内存：8GB RAM 或更高
- 存储空间：至少100MB可用空间

### 2.2 软件环境
- 操作系统：Windows 10/11、macOS 或 Linux
- Python版本：Python 3.8及以上版本
- 开发工具：PyCharm、VS Code或其他Python IDE
- 依赖库：
  - PyQt6：用于构建图形用户界面
  - pyecharts：用于生成各种图表
  - pandas & numpy：用于数据处理
  - requests：用于网络请求
  - pyqt6-webengine：用于在PyQt中嵌入Web内容

## 3. 实训内容

### 3.1 数据采集模块
- 利用网络爬虫技术从招聘网站获取计算机相关岗位数据
- 数据字段包括：岗位名称、公司名称、薪资范围、工作地点、学历要求、工作经验要求等

### 3.2 数据清洗模块
- 处理缺失值、重复值和异常值
- 标准化数据格式，如统一薪资单位、工作年限格式等
- 数据分类和标签化处理

### 3.3 数据可视化模块
实现六个维度的数据可视化分析：

1. **薪资区间分布**：使用桑基图展示不同薪资段的岗位数量分布
2. **地区岗位分布**：使用地理分布图展示不同地区的岗位数量差异
3. **学历要求与薪资关联**：使用饼图对比不同学历对应的薪资区间差异
4. **经验要求分布**：使用折线图展现经验要求的集中情况
5. **公司性质与薪资/岗位数量关联**：使用柱状图对比不同公司性质的岗位数量和薪资水平
6. **岗位类型与关键条件交叉分析**：使用热力图挖掘岗位与学历、薪资、经验的交叉关系

### 3.4 用户界面模块
- 登录注册界面：实现用户身份验证和管理
- 主可视化界面：集成所有图表展示功能
- 交互式操作：支持图表筛选、缩放等交互功能

## 4. 实验步骤

### 4.1 环境搭建
1. 克隆项目代码：
   ```
   git clone https://github.com/YLJ109/infoVisaul.git
   cd infoVisaul
   ```
2. 安装Python 3.8以上版本
3. 安装项目依赖包：
   ```
   pip install -r requirements.txt
   ```

### 4.2 数据准备
1. 运行网络爬虫脚本获取原始数据：
   ```
   python web_crawler/crawler_script.py
   ```
2. 执行数据清洗脚本处理原始数据：
   ```
   python data_cleaning/data_cleaning_script.py
   ```

### 4.3 系统运行
1. 启动应用程序：
   ```
   python app.py
   ```
2. 使用默认账户登录或注册新账户进入系统

### 4.4 功能测试
1. 查看各类可视化图表：
   - 薪资区间分布图
   - 地区岗位分布图
   - 学历要求与薪资关联图
   - 经验要求分布图
   - 公司性质与薪资/岗位数量关联图
   - 岗位类型与关键条件交叉分析图
2. 测试图表交互功能，如缩放、筛选等
3. 验证用户登录和注册功能

### 4.5 结果分析
1. 分析各图表所展示的数据规律
2. 总结计算机相关岗位的就业市场特征
3. 根据分析结果提出职业发展建议
